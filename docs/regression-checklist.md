## 프론트엔드

### 인증/인가
- [ ] 로그인 성공 (정상 자격증명)
- [ ] 로그인 실패 (잘못된 비밀번호)
- [ ] 로그아웃 실행 후 세션 해제
- [ ] 비밀번호 재설정 이메일 발송
- [ ] 로그아웃 상태에서 보호 페이지 접근 차단

### 페이지 로드
- [ ] 메인 페이지 표시 (2초 이내)
- [ ] 상품 목록 표시
- [ ] 상품 상세 페이지 로드
- [ ] 이미지 로드 (broken image 없음)
- [ ] 모바일 반응형 레이아웃

### 검색
- [ ] 키워드 검색 결과 로드
- [ ] 필터 기능 (카테고리, 가격대)
- [ ] 자동완성 작동
- [ ] 검색 결과 없음 표시

### 장바구니
- [ ] 상품 추가/제거
- [ ] 수량 변경
- [ ] 합계 가격 계산 정확
- [ ] 장바구니 색 접근
- [ ] 품절 상품 비활성화

### 결제
- [ ] 주문 확인 페이지 로드
- [ ] 카드 정보 입력 및 검증
- [ ] 주문 성공 후 확인 리다이렉트
- [ ] 주문 완료 이메일 발송
- [ ] 장바구니 비워짐 확인

- [ ] GET /api/products 응답 200

- [ ] API 응답 < 500ms
- [ ] 이미지 캐시 로드
