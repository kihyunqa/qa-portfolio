## 🔴 신규 피드백 — 국민취업제도 담당자 (2025-04-25)

이 4가지를 반드시 처리해야 함. 4번이 핵심.

### 1. 이력서 ≠ 경력기술서 (내용 차별화)
- 현재 두 문서가 거의 동일해서 "같은 거 2개 본 것 같다"는 피드백
- 이력서: 짧고 압축적, 핵심 키워드 위주
- 경력기술서: 맥락·상황·내가 한 일·결과를 풀어서 서술하는 방식으로 차별화 필요
- **아직 미완료. 별도 세션에서 PDF 원본 보고 진행할 것**

### 2. 포트폴리오 이름 색상 → 흰색으로
- intro-banner의 이름(성기현)이 보라색으로 보임
- 흰색(#ffffff)으로 변경 필요
- **미완료**

### 3. 가독성 개선 — 회색·작은 글씨 문제
- 회색 글씨, 작은 글씨가 눈에 잘 안 들어온다는 피드백
- 구체적으로:
  - --muted / --dim 계열 색상 텍스트를 더 밝게 또는 흰색 계열로 조정
  - 소제목·설명 텍스트 font-size 살짝 키우기
  - 특히 타임라인 날짜, 섹션 서브텍스트, 태그 등 작은 요소들 확인
- **미완료**

### 4. 핵심 — "킥"이 없다. 내 색깔이 보여야 한다 (최우선)
- 내용은 많은데 "이 사람이 뭘 잘하는지, 뭘 말하고 싶은 건지"가 안 보임
- 한 방이 필요함. 내 색깔, 나만의 강점이 처음부터 명확하게 드러나야 함
- **방향:**
  - intro-banner 헤드라인/서브텍스트를 다시 써야 함
  - "6.9년 QA + AI 도구 직접 만든 사람" 이 조합이 핵심 킥
  - 단순히 경력 나열이 아니라 "왜 나인가"가 첫 문장부터 느껴져야 함
  - 포트폴리오 전체 흐름도 이 킥을 중심으로 재정비 필요
- **미완료 — 다음 세션 최우선 작업**
