## 핵심 흐름 (이 흐름 절대 깨지 말 것)

경력(배경) → MCP 자동화(무엇을 했나) → 코드 증거(어떻게 했나) → 연동 증거(실제로 됐나) → TC(결과물) → 스킬

이 흐름이 포트폴리오의 스토리. 섹션 추가/제거 시 이 흐름을 깨는지 반드시 확인.
