### CART-010 버그 렬주
> 모바일 Safari에서 장바구니 수량 변경 시 가끔 0으로 리셋됨 → BUG-003로 등록

## 자동화 TC (평시그)

```javascript
// playwright-tests/cart.spec.js 연계
test('CART-001: 상품 장바구니 추가', async ({ page }) => {
  await page.goto('/products/1');
  await page.click('#add-to-cart');
  const badge = page.locator('.cart-badge');
  await expect(badge).toHaveText('1');
});
```
