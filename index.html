→
Claude MCP
→
TC 자동 생성
→
Notion 저장
→
Playwright 실행
→
GitHub 커밋
  • Claude MCP로 자연어 요구사항을 구조화된 테스트케이스로 자동 변환
  • 생성된 TC를 Notion 데이터베이스에 자동 업로드 및 관리
  • Playwright 스크립트 자동 생성 후 브라우저 E2E 테스트 실행
  • 테스트 결과 스크린샷·로그를 GitHub에 자동 커밋하여 이력 관리
