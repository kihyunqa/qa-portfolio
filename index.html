QA 자동화 파이프라인

Claude MCP(Model Context Protocol)는 AI가 로컬 파일시스템, GitHub, 브라우저를 직접 조작하게 해주는 프로토콜입니다. 기능 명세를 입력하면 테스트케이스가 자동 생성되고, 브라우저 테스트 후 스크린샷이 GitHub에 자동으로 올라갑니다. 이 과정에서 코드는 한 줄도 작성하지 않았습니다.

STEP 01
📋
기능 명세 입력
로그인, 결제 등 기능 설명을 Claude에게 전달
→
STEP 02
🤖
TC 자동 생성
해피패스·네거티브·엣지케이스 24건 자동 작성
→
STEP 03
🌐
브라우저 자동 테스트
Playwright MCP로 실제 브라우저 조작 및 스크린샷
→
STEP 04
