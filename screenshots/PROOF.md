## 스크린샷 업로드 방법

### 방법 1 — GitHub 웹에서 직접 업로드 (가장 쉬움)
1. https://github.com/kihyunqa/qa-portfolio/tree/main/screenshots 접속
2. `Add file` → `Upload files` 클릭
3. PNG 파일 드래그 앤 드롭
4. Commit changes 클릭

### 방법 2 — Claude에게 전달
1. 스크린샷을 채팅창에 이미지로 첨부
2. Claude가 github MCP로 바로 커밋
