# E2E 시나리오: 검색 플로우 자동화

> 생성일: 2026-03-30 | 도구: Claude + playwright MCP

---

## 🎯 테스트 목적

검색 입력 → 결과 표시 → 필터링 → 상세 진입까지 검색 플로우 E2E 검증

---

## 🧪 시나리오 1: 정상 검색 플로우

| 단계 | 액션 | 기대결과 |
|------|------|----------|
| 1 | 메인 페이지 접속 | 페이지 로드 완료 |
| 2 | 검색창 클릭 | 포커스 활성화 |
| 3 | "노트북" 입력 | 자동완성 드롭다운 표시 |
| 4 | 엔터 또는 검색 버튼 클릭 | 검색 결과 페이지 이동 |
| 5 | 결과 목록 확인 | 1개 이상 상품 표시 |
| 6 | 첫 번째 결과 클릭 | 상품 상세 페이지 이동 |

**결과:** ✅ PASS

---

## 🧪 시나리오 2: 검색 결과 없음

| 단계 | 액션 | 기대결과 |
|------|------|----------|
| 1 | 검색창에 "xyzxyzxyz123" 입력 | 입력 허용 |
| 2 | 검색 실행 | 검색 결과 페이지 이동 |
| 3 | 결과 없음 메시지 확인 | "검색 결과가 없습니다" 표시 |
| 4 | 추천 검색어 표시 여부 확인 | 추천어 또는 인기 상품 표시 |

**결과:** ✅ PASS

---

## 🧪 시나리오 3: 검색 필터 적용

| 단계 | 액션 | 기대결과 |
|------|------|----------|
| 1 | "노트북" 검색 결과 페이지 | 전체 결과 표시 |
| 2 | 가격 필터 "10만원 이하" 선택 | 필터 적용 |
| 3 | 필터 적용 결과 확인 | 조건에 맞는 상품만 표시 |
| 4 | 브랜드 필터 추가 선택 | 복합 필터 적용 |
| 5 | 필터 초기화 클릭 | 전체 결과 복원 |

**결과:** ✅ PASS

---

## 🧪 시나리오 4: 검색창 특수문자 입력

| 단계 | 액션 | 기대결과 |
|------|------|----------|
| 1 | 검색창에 `<script>alert(1)</script>` 입력 | 입력 허용 |
| 2 | 검색 실행 | XSS 미실행, 텍스트로 처리 |
| 3 | 결과 페이지 정상 로드 확인 | 페이지 정상 표시 |

**결과:** ✅ PASS (보안 검증 완료)
