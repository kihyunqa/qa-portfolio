# E2E Scenario: 회원가입 플로우 (Signup Flow)

## 시나리오 개요

- **시나리오 ID**: E2E-003
- **제목**: 신규 회원가입 전체 플로우
- **주요 기능**: 회원가입, 이메일 인증, 프로필 설정
- **MCP 도구**: Playwright
- **작성일**: 2026-03-30
- **작성자**: kihyunqa

## 사전 조건

- 미등록 이메일 주소
- 테스트 메일 서버 접근 가능

## 테스트 단계

```gherkin
Feature: 회원가입

  Scenario: 정상 회원가입
    Given 사용자가 /signup 페이지에 있다
    When 이름 입력: "테스트유저"
    And 이메일 입력: "newuser@example.com"
    And 비밀번호 입력: "SecurePass123!"
    And 비밀번호 확인 입력: "SecurePass123!"
    And 가입 버튼 클릭
    Then 이메일 인증 미완료 페이지로 이동
    And 인증 메일 수신 확인 메시지 표시

  Scenario: 이메일 인증 완료
    Given 사용자가 이메일 인증 링크를 클릭한다
    Then /dashboard 페이지로 이동
    And 환영 메시지 표시

  Scenario: 중복 이메일 가입 시도
    Given 이미 등록된 이메일로 가입 시도
    Then 오류 메시지 표시: "이미 사용 중인 이메일입니다"
```

## 예상 결과
